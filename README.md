# PyShell

Command line interpreter implemented in Python.

## Supported commands

* cd [dir]
* pwd
* time command
* clr
* dir [dir]
* environ
* echo [comment]
* help
* quit
* bg
* fg
* jobs
* set var val
* shift
* test
* umask [mask]
* unset var

## Features

Redirection: for built-in commands
